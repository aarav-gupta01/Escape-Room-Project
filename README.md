**Escape Room**
**Description**
Escape Room is a text-based game that prompts the user to navigate through a hidden room, solve two randomly chosen password puzzles, and reach the exit.
**How to run**
To run this program, all you need to do is open escape_room.py and click Run.
**Features**
Allows the user to move up, down, left, and right through the room
Alerts the user when there is a wall nearby
Prompts the user to solve two randomized password puzzles
Tracks completed puzzles and requires both to be solved before the user can escape
**Next steps**
If I had more time, I would add a feature that displays the parts of the room the user has already explored. The map would update after every move, making navigation easier while still keeping unexplored areas hidden.
